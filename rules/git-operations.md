# Git Operations

All git operations — commits, branches, PRs, merges, rebases — must follow the **ccupa:git-conventions** skill.

Invoke the ccupa:git-conventions skill before:
- Creating or amending commits
- Naming branches
- Writing PR titles or descriptions
- Deciding between atomic commits and squashing
